# Awards App

## Requirements

- Node V14
- PostgreSQL

## Running App

### Frontend

```
cd awards-fe
```

```
npm install
```

```
npm run serve
```

### Backend

```
cd awards-be
```

```
npm install
```

```
import file schema.sql to postgreSQL
```

```
create file .env based on .env.example
```

```
npm start
```
